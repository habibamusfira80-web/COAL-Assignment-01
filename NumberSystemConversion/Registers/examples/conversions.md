# Number System Conversion Examples

## Binary to Decimal
- (1011)₂ = 1×2³ + 0×2² + 1×2¹ + 1×2⁰ = 8+0+2+1 = (11)₁₀
- (11010)₂ = 1×2⁴ + 1×2³ + 0×2² + 1×2¹ + 0×2⁰ = 16+8+0+2+0 = (26)₁₀

## Decimal to Binary
- (13)₁₀ = 13÷2=6 R1 ↑ = (1101)₂
         6÷2=3 R0 |
         3÷2=1 R1 |
         1÷2=0 R1 |
- (25)₁₀ = (11001)₂

## Decimal to Hexadecimal
- (255)₁₀ = 255÷16=15 R15(F) ↑ = (FF)₁₆
          15÷16=0  R15(F) |
- (425)₁₀ = (1A9)₁₆

## Hexadecimal to Decimal
- (1A)₁₆ = 1×16¹ + A×16⁰ = 16 + 10 = (26)₁₀
- (2F)₁₆ = 2×16¹ + F×16⁰ = 32 + 15 = (47)₁₀

## Binary to Hexadecimal
- (101111)₂ = 10 1111 = 2F = (2F)₁₆
- (11011011)₂ = 1101 1011 = DB = (DB)₁₆

## Hexadecimal to Binary
- (3B)₁₆ = 3 → 0011, B → 1011 = (00111011)₂
- (A7)₁₆ = A → 1010, 7 → 0111 = (10100111)₂
